# Weather Dashboard 🌤️

A high-performance, responsive weather application built to deliver real-time meteorological data. This project is a cloud-native implementation demonstrating proficiency in modern frontend architecture, asynchronous API integration, and strict type safety.

## ✨ Key Features

* **Real-Time Data Fetching:** Integrates with the OpenWeatherMap API to provide live temperature, humidity, and wind conditions.
* **Dynamic City Search:** Allows users to query weather data for any global city with instant UI updates.
* **Strict Type Safety:** Built entirely with TypeScript to ensure robust data handling and prevent runtime errors.
* **Responsive UI:** A mobile-first, dashboard-style interface crafted with Tailwind CSS for seamless viewing across all devices.
* **Optimized Performance:** Leverages the Next.js App Router for efficient client-side rendering and state management.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **API:** [OpenWeatherMap API](https://openweathermap.org/)
* **Hosting:** [Vercel](https://vercel.com/)

## 🌐 Deployment to Vercel

This project is designed to be deployed directly from GitHub to Vercel.

1.  **Import:** Connect your GitHub account to Vercel and import the `weather-dashboard` repository.
2.  **Environment Variables:** In the Vercel dashboard, navigate to **Settings > Environment Variables**.
3.  **Add Key:** Add a new variable with the key `NEXT_PUBLIC_OPENWEATHER_API_KEY` and your actual API key as the value.
4.  **Deploy:** Vercel will automatically detect the Next.js configuration and build your live site.
